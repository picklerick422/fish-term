# wand-term P2 Implementation Plan (Driver abstraction: Local PTY + SSH + DriverFactory)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the `TerminalDriver` abstraction across all three transports — refactor the existing native shell into `LocalPtyDriver`, add a profile-driven password-only `SshDriver` (backed by a new direct native SSH entrypoint), and add a `DriverFactory` + transport picker so a single session can run over wand-agent, local PTY, or SSH.

**Architecture:** P1 already defined `TerminalDriver` and `WandWebSocketDriver`. P2 makes the two native transports implement the same interface by wrapping `libexample_driver.so`. The native is extended with one new function, `startSsh(host,port,user,password,cols,rows)`, which calls the already-implemented `SSHSession::connect(...)` directly (the in-band `ssh user@host` + inline-password flow is bypassed). `DriverFactory` maps a transport spec to a driver; `TerminalSession` (unchanged from P1) wires whichever driver to the controller.

**Tech Stack:** ArkTS / ArkUI, HarmonyOS NDK (C++ napi, **native is modified this phase**), libssh2/mbedTLS (already linked).

**Spec:** `docs/superpowers/specs/2026-06-16-wand-harmony-terminal-design.md` §11 P2, §4.1, §4.4. Decision (2026-06-16): SSH is **password-only**; SSH gets a clean native entrypoint (no in-band trigger).

**Depends on:** P0+P1 (`docs/superpowers/plans/2026-06-16-wand-term-p0-p1.md`) merged/verified.

## Constraints & decisions (read first)

- **Native is modified this phase.** Unlike P0/P1, P2 edits `entry/src/main/cpp/example_driver.cpp`. No new dependencies are introduced — `ssh_session.cpp` + libssh2 + mbedTLS are already compiled and linked, so `entry/third_party/{mbedtls,libssh2}` must be present (already required by the existing build). DevEco's CMake rebuilds `libexample_driver.so` automatically; **no Docker** is needed (Docker is only for the `libghostty_vt.a` prebuilt, which is unchanged).
- **Singleton native = one native-backed session at a time.** `GetDriver()` is a process-wide singleton with a single output channel. `LocalPtyDriver` and `SshDriver` both wrap it, so only **one** local-or-ssh session can be active concurrently. This is fine for P2 (single session). **P3 (multi-tab) must resolve this** — e.g., a handle/instance-based native API — before allowing multiple native-backed tabs. wand-agent tabs are unaffected (each is its own WebSocket, pure ArkTS).
- **`startSsh` connect is synchronous** (mirrors the existing in-band flow): `SSHSession::connect` blocks the JS thread during TCP+auth. `SshDriver` emits `Connecting` before calling it. Moving connect to a worker is a later refinement, noted but out of scope.
- **Native output is delivered as JS strings** via the existing threadsafe callback (not byte frames), so the native drivers feed `controller.feed` directly — no `Utf8Framer` (that's wand-only, where frames arrive as `ArrayBuffer`).
- **Git** still runs on a native FS via `tools/git-init.sh` (the `/mnt/linux_share` chmod limitation). Commit steps below are the intended grouping; run them where git works.

```cpp
  bool StartSsh(const std::string& host, int port, const std::string& user,
                const std::string& password, int cols, int rows)
  {
      Stop();  // tear down any existing local/ssh session and threads

      {
          std::lock_guard<std::mutex> lock(m_mutex);
          m_cols = cols > 0 ? cols : DEFAULT_COLS;
          m_rows = rows > 0 ? rows : DEFAULT_ROWS;
          m_mode = Mode::RemoteSsh;
          m_currentLine.clear();
          m_pendingPassword.clear();
      }

      auto session = std::make_unique<SSHSession>();
      session->setOutputCallback([this](const std::string& output) {
          PushOutput(output);
      });

      std::string error;
      if (!session->connect(host, port > 0 ? port : 22, user, password, m_cols, m_rows, error)) {
          PushOutput("SSH connection failed: " + error + "\r\n");
          std::lock_guard<std::mutex> lock(m_mutex);
          m_mode = Mode::LocalShell;
          return false;
      }

      {
          std::lock_guard<std::mutex> lock(m_mutex);
          m_sshSession = std::move(session);
          m_mode = Mode::RemoteSsh;
          m_running = true;
      }
      return true;
  }
```

napi_value StartSsh(napi_env env, napi_callback_info info)
{
    size_t argc = 6;
    napi_value args[6] = {nullptr, nullptr, nullptr, nullptr, nullptr, nullptr};
    napi_get_cb_info(env, info, &argc, args, nullptr, nullptr);

    std::string host = argc >= 1 ? ReadStringArg(env, args[0]) : std::string();
    int32_t port = 22;
    if (argc >= 2) {
        napi_get_value_int32(env, args[1], &port);
    }
    std::string user = argc >= 3 ? ReadStringArg(env, args[2]) : std::string();
    std::string password = argc >= 4 ? ReadStringArg(env, args[3]) : std::string();
    int32_t cols = DEFAULT_COLS;
    int32_t rows = DEFAULT_ROWS;
    if (argc >= 5) {
        napi_get_value_int32(env, args[4], &cols);
    }
    if (argc >= 6) {
        napi_get_value_int32(env, args[5], &rows);
    }

    const bool ok = GetDriver().StartSsh(host, port, user, password, cols, rows);
    napi_value result;
    napi_get_boolean(env, ok, &result);
    return result;

```typescript
import { DriverState, DriverStatus, TerminalDriver } from './TerminalDriver';
import exampleDriver from 'libexample_driver.so';

interface ExampleDriverNativeBridge {
  initialize: (filesDir: string) => void;
  start: (cols: number, rows: number) => boolean;
  startSsh: (host: string, port: number, user: string, password: string, cols: number, rows: number) => boolean;
  stop: () => void;
  writeInput: (data: string) => void;
  drainOutput: () => string;
  setOutputCallback: (callback: ((data: string) => void) | null) => void;
  resize: (cols: number, rows: number) => void;
}

const nativeBridge = exampleDriver as ExampleDriverNativeBridge;

// Local shell over the bundled native PTY (fish). Wraps the singleton
// libexample_driver.so. Only one native-backed driver may be active at a time.
export class LocalPtyDriver implements TerminalDriver {
  private readonly filesDir: string;
  private outputCb: ((data: string) => void) | null = null;
  private statusCb: ((s: DriverStatus) => void) | null = null;
  private started: boolean = false;

  constructor(filesDir: string) {
    this.filesDir = filesDir;
  }

  onOutput(cb: (data: string) => void): void {
    this.outputCb = cb;
  }

  onStatus(cb: (s: DriverStatus) => void): void {
    this.statusCb = cb;
  }

  start(cols: number, rows: number): void {
    this.emit(DriverState.Connecting);
    nativeBridge.initialize(this.filesDir);
    nativeBridge.setOutputCallback((data: string): void => {
      if (data.length > 0 && this.outputCb) {
        this.outputCb(data);
      }
    });
    this.started = nativeBridge.start(cols, rows);
    if (this.started) {
      this.emit(DriverState.Connected);
    } else {
      this.emit(DriverState.Error, 'failed to start local shell');
    }
  }

  write(data: string): void {
    if (this.started && data.length > 0) {
      nativeBridge.writeInput(data);
    }
  }

  resize(cols: number, rows: number): void {
    if (this.started) {
      nativeBridge.resize(cols, rows);
    }
  }

  stop(): void {
    nativeBridge.setOutputCallback(null);
    if (this.started) {
      nativeBridge.stop();
      this.started = false;
    }
    this.emit(DriverState.Closed);
  }

  private emit(state: DriverState, message?: string): void {
    if (this.statusCb) {
      this.statusCb({ state, message });
    }
  }
}
```

import { DriverState, DriverStatus, SshConnectConfig, TerminalDriver } from './TerminalDriver';
import exampleDriver from 'libexample_driver.so';

interface ExampleDriverNativeBridge {
  initialize: (filesDir: string) => void;
  start: (cols: number, rows: number) => boolean;
  startSsh: (host: string, port: number, user: string, password: string, cols: number, rows: number) => boolean;
  stop: () => void;
  writeInput: (data: string) => void;
  drainOutput: () => string;
  setOutputCallback: (callback: ((data: string) => void) | null) => void;
  resize: (cols: number, rows: number) => void;
}

const nativeBridge = exampleDriver as ExampleDriverNativeBridge;

// SSH (password auth) over libssh2 via the native startSsh entrypoint. Wraps the
// singleton libexample_driver.so; only one native-backed driver active at a time.
// Note: native connect is synchronous and briefly blocks while authenticating.
export class SshDriver implements TerminalDriver {
  private readonly config: SshConnectConfig;
  private outputCb: ((data: string) => void) | null = null;
  private statusCb: ((s: DriverStatus) => void) | null = null;
  private started: boolean = false;

  constructor(config: SshConnectConfig) {
    this.config = config;
  }

  onOutput(cb: (data: string) => void): void {
    this.outputCb = cb;
  }

  onStatus(cb: (s: DriverStatus) => void): void {
    this.statusCb = cb;
  }

  start(cols: number, rows: number): void {
    this.emit(DriverState.Connecting);
    nativeBridge.setOutputCallback((data: string): void => {
      if (data.length > 0 && this.outputCb) {
        this.outputCb(data);
      }
    });
    this.started = nativeBridge.startSsh(
      this.config.host,
      this.config.port > 0 ? this.config.port : 22,
      this.config.user,
      this.config.password,
      cols,
      rows);
    if (this.started) {
      this.emit(DriverState.Connected);
    } else {
      this.emit(DriverState.Error, 'SSH connection failed');
    }
  }

  write(data: string): void {
    if (this.started && data.length > 0) {
      nativeBridge.writeInput(data);
    }
  }

  resize(cols: number, rows: number): void {
    if (this.started) {
      nativeBridge.resize(cols, rows);
    }
  }

  stop(): void {
    nativeBridge.setOutputCallback(null);
    if (this.started) {
      nativeBridge.stop();
      this.started = false;
    }
    this.emit(DriverState.Closed);
  }

  private emit(state: DriverState, message?: string): void {
    if (this.statusCb) {
      this.statusCb({ state, message });
    }
  }
}

```typescript
import { DriverSpec, TerminalDriver, TransportType } from './TerminalDriver';
import { WandWebSocketDriver } from './WandWebSocketDriver';
import { LocalPtyDriver } from './LocalPtyDriver';
import { SshDriver } from './SshDriver';

// Builds a TerminalDriver from a transport spec. `filesDir` is required for the
// local PTY transport (native shell initialization).
export function createDriver(spec: DriverSpec, filesDir: string): TerminalDriver {
  switch (spec.transport) {
    case TransportType.Wand:
      if (!spec.wand) {
        throw new Error('wand config missing for Wand transport');
      }
      return new WandWebSocketDriver(spec.wand);
    case TransportType.Ssh:
      if (!spec.ssh) {
        throw new Error('ssh config missing for Ssh transport');
      }
      return new SshDriver(spec.ssh);
    case TransportType.LocalPty:
      return new LocalPtyDriver(filesDir);
    default:
      throw new Error('unknown transport: ' + spec.transport);
  }
}
```

import { common } from '@kit.AbilityKit';
import { TerminalSurface } from 'libghostty_ohos';
import { TerminalSession } from '../session/TerminalSession';
import { createDriver } from '../transport/DriverFactory';
import { DriverState, DriverStatus, DriverSpec, TransportType } from '../transport/TerminalDriver';

function stateLabel(s: DriverStatus): string {
  switch (s.state) {
    case DriverState.Connecting:
      return 'Connecting…';
    case DriverState.Connected:
      return 'Connected';
    case DriverState.Reconnecting:
      return 'Reconnecting…';
    case DriverState.Closed:
      return 'Closed';
    case DriverState.Error:
      return 'Error: ' + (s.message ?? '');
    default:
      return 'Idle';
  }
}

@Entry
@Component
struct Index {
  @State private connected: boolean = false;
  @State private transport: TransportType = TransportType.Wand;
  // wand
  @State private host: string = '192.168.1.10';
  @State private port: string = '8765';
  @State private token: string = '';
  @State private tls: boolean = false;
  // ssh
  @State private sshHost: string = '192.168.1.10';
  @State private sshPort: string = '22';
  @State private sshUser: string = 'root';
  @State private sshPassword: string = '';
  @State private statusText: string = 'Idle';

  private session: TerminalSession | null = null;
  private context: common.UIAbilityContext | null = null;

  aboutToAppear(): void {
    this.context = this.getUIContext().getHostContext() as common.UIAbilityContext;
  }

  aboutToDisappear(): void {
    this.teardown();
  }

  private buildSpec(): DriverSpec {
    if (this.transport === TransportType.Wand) {
      return {
        transport: TransportType.Wand,
        wand: {
          host: this.host.trim(),
          port: Number(this.port) || 8765,
          token: this.token.trim(),
          tls: this.tls
        }
      };
    }
    if (this.transport === TransportType.Ssh) {
      return {
        transport: TransportType.Ssh,
        ssh: {
          host: this.sshHost.trim(),
          port: Number(this.sshPort) || 22,
          user: this.sshUser.trim(),
          password: this.sshPassword
        }
      };
    }
    return { transport: TransportType.LocalPty };
  }

  private connect(): void {
    if (!this.context) {
      return;
    }
    const filesDir: string = this.context.filesDir;
    const driver = createDriver(this.buildSpec(), filesDir);
    const session: TerminalSession = new TerminalSession(driver);
    session.onStatus((s: DriverStatus): void => {
      this.statusText = stateLabel(s);
    });
    session.attach(this.context);
    this.session = session;
    this.connected = true;
  }

  private teardown(): void {
    if (this.session) {
      this.session.detach();
      this.session = null;
    }
  }

  private disconnect(): void {
    this.teardown();
    this.connected = false;
    this.statusText = 'Idle';
  }

  build() {

      if (!this.connected) {
        this.buildForm();
      } else {
        this.buildTerminal();
      }
    }
    .width('100%')
    .height('100%')
    .backgroundColor('#0B0D10')
  }
